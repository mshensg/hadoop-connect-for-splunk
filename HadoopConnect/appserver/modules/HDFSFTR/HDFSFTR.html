This app needs to be configured!

You will need the following:

  • Install and configure Hadoop comand line utilities (including Java if necessary)
  • Ensure that the following command works: $HADOOP_HOME/bin/hadoop fs -ls hdfs://<namenode>:<port>/
  • Configure the app by adding Hadoop clusters to it
  • If you need a secure connection you will need a Kerberos principal name and associated keytab file.
